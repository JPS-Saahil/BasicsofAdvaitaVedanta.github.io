# Basics of Advaita Vedanta

An interactive visual exploration of Advaita Vedānta presented through diagrams, guided explanations, animations, and structured study.

This project is designed to make core non-dual teachings easier to approach for modern learners while preserving the depth of the original philosophy.

## Overview

Advaita Vedānta is a school of Indian philosophy centered on non-duality, the nature of the Self, and the relationship between Ātman and Brahman.

This project presents these ideas in a visual and interactive form. Instead of relying only on text, the website uses diagrams, navigation flow, and simple interaction patterns to help users study the material step by step.

The project is intended for:

- Students of philosophy
- Beginners exploring Advaita Vedānta
- Self-learners
- Teachers and discussion groups
- Anyone interested in reflective study of non-dual thought

## Features

- Chapter-based learning structure

## Local Development

Clone the repository:

```bash
git clone https://github.com/JPS-Saahil/BasicsofAdvaitaVedanta.github.io.git
cd BasicsofAdvaitaVedanta.github.io
```

Open the project locally by serving the files with any static server, or open the main HTML file directly in your browser.

Example using Python:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deployment

This project is deployed using GitHub Pages.

To update the live site:

1. Push changes to the repository.
2. Verify the main branch contains the latest version.
3. Check the GitHub Pages deployment settings.
4. Confirm the public site loads correctly after deployment.
